## 📝 A CLI based todo application
Made to manage simple tasks easily in the terminal

## 🚀 Getting started

### Prerequisites
- Python  (3.13.9 or higher)

### Installation

1. Clone the repository
```bash
git clone https://github.com/Jyn-Tuyor/cli-todo-app.git
```
2. Install dependencies
```bash
pip install tabulate
```
```bash
pip install -r requirements.txt 
```

### Running the app

1. Usage
```bash
python main.py [options]
```
2. Usage examples
```bash
python main.py add "do homework"
python main.py list
python main.py done 1
python main.py delete 1
python main.py clear
```
